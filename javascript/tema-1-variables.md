# TEMA 1: Variables y Tipos de Datos

Las variables son como cajas donde guardamos información en JavaScript.

## Tipos de variables
* **let**: Puede cambiar su valor.
* **const**: Su valor es fijo y no cambia.

## Ejemplo de código
```javascript
let nombre = "Ana"; // Esto es un String (texto)
const edad = 25;    // Esto es un Number (número)
